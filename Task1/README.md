# Modern Course Dashboard

A simple education dashboard mockup built with HTML and CSS.

## Files

- `index.html` — page structure for the dashboard
- `style.css` — responsive styling for the header, course cards, and sidebar
- `image.png` — example or placeholder image asset

## Overview

This project includes:

- sticky top navigation
- hero dashboard section
- course cards with hover effects
- instructor and upcoming tasks sidebar
- mobile-friendly layout

## Usage

1. Open `index.html` in a browser.
2. Make sure `style.css` is in the same folder.
3. Replace placeholder images like `logo.png`, `image.png`, `image1.jpg`, `image2.jpg`, and `image3.jpg` with your own assets.

## Notes

- No build tools are required.
- This is a static frontend prototype suitable for learning dashboard UI concepts.
