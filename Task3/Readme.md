# Task 3 - User Management Dashboard

## Overview
This project is a small interactive dashboard for managing a user directory using vanilla HTML, CSS, and JavaScript.
It demonstrates DOM manipulation, form handling, live search filtering, and dynamic UI updates.

## Features
- Add a new user with name and email
- Delete users from the directory
- Live search filter by name or email
- Real-time user count display
- Responsive layout for desktop and mobile

## Files
- `index.html` - dashboard structure and layout
- `style.css` - responsive styling and component design
- `script.js` - DOM updates, event handling, and data logic

## Usage
1. Open `index.html` in a browser.
2. Add a user by entering a full name and email address.
3. Use the search field to filter users instantly.
4. Click **Delete** to remove a user.

## Notes
- This dashboard uses an in-memory user list; data is not persisted after reload.
- The app includes basic input sanitization to keep rendered user details safe.
